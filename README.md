# Student Course Management System

This is a Spring Boot application that manages Students and Courses with CRUD operations.

## Features
- Create Student
- View Students with Courses
- Update Student
- Many-to-Many Relationship
- Custom JOIN Query

## Technologies Used
- Spring Boot
- Spring Data JPA
- H2 Database
- JSP
- Maven

## How to Run
1. Clone the repository
2. Run the application
3. Open http://localhost:8080/students
